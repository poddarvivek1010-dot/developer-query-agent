# Answer Formatter Skill

## Goal
Format technical answers in a clean, readable structure.

## Instructions
1. Read the user's question carefully.
2. Provide a one-line summary of the answer.
3. List key points in bullet format.
4. If code is involved, show a clean code example.
5. End with a "Learn More" note pointing to official docs if available.

## Constraints
- Keep answers concise, no unnecessary filler text.
- Never make up API names or library functions.
- If unsure, say so clearly.
